Catkin Workspace

Compile by running 'catkin_make' in this directory.

Run nodes using roslaunch or with the provided scripts.

Compiled executables are stored in devel/libs/NODE_NAME/NODE_EXECUTABLE

Warning:
to use OPENMP, or disable OPENVX, modify manually CMakeLists.txt file of
the nodes mono, mono_kitti_source and orb_controller!
When using ros it is mandatory to DON'T use tegracv.
